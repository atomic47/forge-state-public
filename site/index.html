§ 08 · The sandbox

Disposable. Capped. Reproducible.

Container · per repo · disposable
Docker, deny-egress, capped.
  • ▸Per-language base: node:20 · python:3.12 · go:1.22 · rust:1.79 · ubuntu:24.04
  • ▸Egress: package registries only (deny-default; no host secrets, no open internet)
  • ▸Caps: cpu=4 · mem=8g · per_repo_timeout_s=1800 · night_budget_s=14400
  • ▸Fresh container per repo; torn down after results are read. No reuse.
build/env.json — the reproducibility anchor
{
  "image_base": "rust:1.79",
  "image_digest": "sha256:b1a4…e9c2",
  "runtimes": { "rustc": "1.79.0",
                "cargo": "1.79.0" },
  "commands": [ "cargo build --release" ],
  "exit_code": 0,
  "duration_s": 142,
  "os": "linux/amd64"
}
The gist embeds this verbatim. A stranger on a different machine can rerun the exact build — same image, same commands, same outcome.

Reproducibility is the anchor. Without env.json, the gist is a story; with it, the gist is a recipe.
